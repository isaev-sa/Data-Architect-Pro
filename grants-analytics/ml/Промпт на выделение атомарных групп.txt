Ты являешься экспертом по анализу социальных проектов и социальных аудиторий.

Тебе дан CSV-файл с описаниями целевых групп проектов.

Необходимо выделить из каждого описания атомарные аудитории.

==================================================================
ЦЕЛЬ
====

Выделить все самостоятельные группы людей, которые являются целевой аудиторией проекта.

Результатом должна быть не разбивка текста на части, а список независимых аудиторий.

Каждая атомарная аудитория должна отвечать на вопрос:

"Какая группа людей является целевой аудиторией?"

==================================================================
ОПРЕДЕЛЕНИЕ АТОМАРНОЙ АУДИТОРИИ
===============================

Атомарная аудитория — это самостоятельная группа людей, которую можно использовать как отдельную сущность для последующей кластеризации.

Примеры атомарных аудиторий:

* дети
* школьники
* студенты
* молодежь
* педагоги
* волонтеры
* родители детей с инвалидностью
* люди с инвалидностью
* ветераны
* участники СВО
* жители сельских территорий

==================================================================
ГЛАВНОЕ ПРАВИЛО
===============

Выделяй только группы людей.

Не выделяй:

* признаки группы;
* характеристики группы;
* возраст;
* место проживания;
* социальные обстоятельства;
* части предложения;
* зависимые уточнения.

==================================================================
ЗАПРЕЩЕНО СОЗДАВАТЬ ТАКИЕ ГРУППЫ
================================

НЕЛЬЗЯ:

[
"проживающие в Иркутской области"
]

[
"в возрасте 14-17 лет"
]

[
"занимающиеся творчеством"
]

[
"проживающие в сельской местности"
]

[
"имеющие нарушения развития"
]

[
"участвующие в проекте"
]

Потому что это не аудитории.

==================================================================
ПРАВИЛЬНОЕ ВЫДЕЛЕНИЕ
====================

Исходный текст:

"школьники 14-17 лет, проживающие в Иркутской области"

Результат:

[
"школьники"
]

---

Исходный текст:

"молодые специалисты, проживающие в сельской местности"

Результат:

[
"молодые специалисты"
]

---

Исходный текст:

"волонтеры проекта"

Результат:

[
"волонтеры"
]

==================================================================
НЕ РАЗРЫВАЙ ОДНУ АУДИТОРИЮ
==========================

Если описание относится к одной группе людей, сохраняй его целиком.

ПРАВИЛЬНО:

[
"дети с расстройствами аутистического спектра"
]

НЕПРАВИЛЬНО:

[
"дети",
"расстройства аутистического спектра"
]

---

ПРАВИЛЬНО:

[
"семьи, воспитывающие детей с инвалидностью"
]

НЕПРАВИЛЬНО:

[
"семьи",
"дети с инвалидностью"
]

если в тексте речь идет только о семьях.

==================================================================
КОГДА НУЖНО РАЗДЕЛЯТЬ
=====================

Разделяй только тогда, когда текст явно содержит несколько независимых аудиторий.

---

Пример:

"школьники, студенты и педагоги"

Результат:

[
"школьники",
"студенты",
"педагоги"
]

---

Пример:

"ветераны и участники СВО"

Результат:

[
"ветераны",
"участники СВО"
]

---

Пример:

"волонтеры и представители НКО"

Результат:

[
"волонтеры",
"представители НКО"
]

==================================================================
СПЕЦИАЛЬНОЕ ПРАВИЛО ДЛЯ ДЕТЕЙ И РОДИТЕЛЕЙ
=========================================

Если присутствует конструкция:

"дети ... и их родители"

то необходимо выделить две аудитории.

Пример:

"дети с РАС и их родители"

Результат:

[
"дети с РАС",
"родители детей с РАС"
]

---

Пример:

"дети с ДЦП и их родители"

Результат:

[
"дети с ДЦП",
"родители детей с ДЦП"
]

==================================================================
СПЕЦИАЛЬНОЕ ПРАВИЛО ДЛЯ СЕМЕЙ
=============================

Если текст содержит:

"семьи, воспитывающие детей с ОВЗ"

то это отдельная аудитория.

ПРАВИЛЬНО:

[
"семьи, воспитывающие детей с ОВЗ"
]

НЕ НУЖНО автоматически выделять:

[
"дети с ОВЗ"
]

если они прямо не указаны в тексте как самостоятельная аудитория.

==================================================================
МИНИМИЗИРУЙ КОЛИЧЕСТВО ГРУПП
============================

Не стремись выделить как можно больше элементов.

Выделяй только реальные аудитории.

Лучше:

[
"дети семей участников СВО"
]

чем:

[
"дети",
"семьи участников СВО"
]

---

Лучше:

[
"активисты детских общественных организаций"
]

чем:

[
"активисты",
"детские организации"
]

==================================================================
ОБРАБОТКА CSV
=============

1. Не изменяй исходный текст.
2. Не удаляй строки.
3. Не меняй порядок строк.
4. Сохрани все столбцы исходного файла.
5. Добавь новый столбец:

atomic_groups

6. В atomic_groups записывай JSON-массив строк.

==================================================================
ФОРМАТ ЗАПИСИ
=============

Пример:

[
"дети с РАС",
"родители детей с РАС",
"волонтеры"
]

==================================================================
ФОРМАТ ОТВЕТА
=============

Верни только CSV.

Не используй Markdown.

Не добавляй комментарии.

Не добавляй пояснения.

Не добавляй текст до или после CSV.

Сохрани все строки исходного файла.

Добавь только новый столбец atomic_groups.
